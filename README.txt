Setup and Installation Process : 
    1. Download the file and extract the file in your local system
    2. Open the extract file folder double tab the index.html file it's automatically open default browser or select the other web browser
    3. All html pages are linked in index.html file so once you open the index.html in your browser all pages are linked. If you open the individual html file double tab the .html file
    4. Finally you view the HTML design view in your local system browser
